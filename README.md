# Cys-pKa-MD-calculations
A series of scripts to automate replica-exchange thermodynamic integration molecular dynamics (RETI-MD) calculations for Cys pKa using the Gromacs MD engine and PBS job scheduler. 

Methodology employed in these papers: 
  - [Awoonor-Williams and Rowley, *JCTC*, **2016**, 12 (9), 4662-73.](https://pubs.acs.org/doi/10.1021/acs.jctc.6b00631)
  - [Awoonor-Williams and Rowley, *JCIM*, **2018**, 58(9), 1935-46.](https://pubs.acs.org/doi/10.1021/acs.jcim.8b00454)
